# Birthday Greeting Web Project

This project is a simple birthday greeting web application that allows users to navigate between different pages and replace an image on the main page.

## Project Structure

```
birthday-greeting
├── index.html          # Main page for the birthday greeting
├── pages               # Contains additional pages
│   ├── page1.html      # Content for the first page
│   └── page2.html      # Content for the second page
├── css                 # Contains styles for the project
│   └── styles.css      # Styles for the main page and buttons
├── js                  # Contains JavaScript functionality
│   └── main.js         # JavaScript for handling interactions
├── assets              # Directory for assets
│   └── .gitkeep        # Keeps the assets directory in version control
└── README.md           # Documentation for the project
```

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to view the main page.
3. Use the buttons to navigate to `page1.html` and `page2.html`.
4. To replace the image on the main page, update the `src` attribute of the image tag in `index.html`.
